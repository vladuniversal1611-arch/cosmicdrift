# 🐉 Dragon Merge Blast

Сучасна мобільна гра жанру **«3 в ряд»** для Android, де поєднання кристалів
заряджає **яйця драконів**, а вилуплені дракони **автоматично допомагають** на
полі бою. Виглядає дорожче, цікавіше та сучасніше за класичні «Самоцвіти».

> Готова до запуску гра на HTML5/Canvas, упакована для Android через **Capacitor**
> (готовність до експорту в Google Play — AAB/APK).

---

## ✨ Можливості

### Основна механіка
- Поєднання 3+ кристалів (свайп сусідніх).
- Каскади та **комбо** з ростом множника очок і наростаючим звуком.
- **Особливі кристали** з великих збігів: лінія (4 в ряд) та веселка (5 в ряд).
- **Комбінації особливих кристалів** при свайпі один на одного: лінія+лінія = хрест,
  бомба+бомба = вибух 5×5, веселка+лінія/бомба та веселка+веселка = очищення поля.
- Кожен збіг генерує **енергію**, що заряджає драконів і яйця.

### 🧪 Бустери та полірування
- **Бустери в бою**: 🔨 молот (розбити кристал), 🔀 перемішування, ➕5 ходів
  (купуються в магазині за золото).
- **Інтерактивний туторіал** на 1-му рівні з покроковими підказками.
- **Підказка ходу**: поле підсвічує можливий хід, якщо гравець вагається.
- **Вібро-відгук (haptics)** на збіги/комбо/перемогу (вмикається в налаштуваннях).
- «Соковиті» пружні анімації кристалів, ударні хвилі та ефекти частинок.

### 🐲 Дракони (допомагають автоматично)
| Дракон | Здатність |
|--------|-----------|
| 🐉 **Flare** — Вогняний | Спалює цілі ряди кристалів |
| 🐲 **Frost** — Крижаний | Заморожує/розбиває кригу та перешкоди |
| 🦅 **Storm** — Грозовий | Б’є блискавками по випадкових кристалах |
| 🦎 **Verdant** — Лісовий | Створює бонусні кристали-підсилювачі |
| ✨ **Aether** — Ефірний | Перемішує поле та дарує додаткові ходи |

Дракон заряджається від збігів; коли смужка заповнюється — **здатність
спрацьовує сама** (раз за хід). Сила залежить від рівня прокачки.

### ❤️ Життя та боси
- **Система життів/сердець**: 5 сердець, регенерація 1 шт. / 20 хв; невдала спроба
  коштує серце; поповнення рекламою або за 💎 (класичний F2P-цикл).
- **Реальні боси** наприкінці кожного острова: ❤ HP-бар, дамаг від збігів і
  драконів, а кожні 5 ходів **бос контратакує** — додає кригу на поле.

### 📊 Соціальне
- **Локальна таблиця рекордів** за зірками (з імітованими суперниками).
- **Жива анімація острова**: дракони махають крилами, літаючий дракон, хмари, іскри.

### 🌐 Локалізація
- **6 мов**: українська, English, Español, Deutsch, Français, Português.
- Перемикач мови в налаштуваннях; увесь інтерфейс перекладається миттєво.

### 🧩 Перешкоди та «сяйво» рівня
- Різноманітні перешкоди: 🧊 крига, 📦 ящики (2 удари), 🔒 заблоковані ланцюгами
  кристали, 🟪 желе (нова ціль «очистити желе»).
- **Заставка рівня** з ціллю на старті та фінальний каскад-феєрверк
  «Sugar Crush» при перемозі з бонусом за невитрачені ходи.

### 🏝️ Прогресія
- **125 рівнів** (> 100) на **5 островах** з різними темами.
- П’ять типів цілей: очки, збір кристалів, крига, **желе** та **бій з босом**.
- **Острів драконів** з інкубатором яєць між рівнями.
- **Колекція** та **прокачування** драконів.
- Поступове **відкриття нових островів**.
- **Щоденні нагороди** (7-денний цикл зі стріками).
- **Досягнення** та **щоденні квести**.

### 💎 Монетизація
- **Реклама за винагороду** (готовий хук під AdMob, з демо-симуляцією).
- **Бойовий пропуск** (50 рівнів, безкоштовна + преміум доріжки).
- **Косметичні скіни драконів** у магазині.
- IAP-плейсхолдери для пакетів кристалів/золота (під Google Play Billing).

### 🎨 Дизайн і техніка
- Яскрава мультяшна графіка, намальована на Canvas (градієнти, глянець).
- Анімації збігів/падінь, **ефекти частинок**, тряска екрана, проліт драконів.
- Преміальний UI, **вертикальна орієнтація**.
- **Збереження** прогресу (localStorage).
- **Звуки та музика** повністю синтезовані через WebAudio (без важких ассетів).
- Плавна робота на мобільних (адаптивний canvas, DPR-масштабування).

## 📁 Структура проєкту

```
www/                 # веб-гра (webDir для Capacitor)
  index.html
  manifest.webmanifest
  css/style.css
  js/data.js         # дракони, 125 рівнів, квести, пропуск, магазин
  js/save.js         # система збережень
  js/audio.js        # WebAudio: SFX + процедурна музика
  js/engine.js       # рушій match-3 (логіка, анімації, частинки, дракони)
  js/ui.js           # усі мета-екрани (DOM)
  js/main.js         # ігровий цикл, HUD, навігація
  assets/            # іконки PWA/застосунку
capacitor.config.json
package.json
server.js            # локальний попередній перегляд
android-resources/   # генератор іконок + тести рушія
```

## 🤖 Збірка APK/AAB для Android (Google Play)

Потрібні: **Node.js 18+**, **JDK 17**, **Android Studio** (Android SDK + Gradle).

```bash
# 1. Встановити залежності
npm install

# 2. Додати Android-платформу (створює теку android/)
npx cap add android

# 3. Синхронізувати веб-гру в Android-проєкт
npx cap sync android

# 4а. Відкрити в Android Studio та зібрати
npx cap open android

# 4б. Або зібрати з командного рядка:
npm run android:build      # debug APK → android/app/build/outputs/apk/debug/
npm run android:release    # release AAB → android/app/build/outputs/bundle/release/
```

### Іконки застосунку
Іконку 1024×1024 згенеровано в `android-resources/icon-1024.png`. Після
`cap add android` згенеруйте mipmap-набір (Android Studio → *Image Asset*,
або плагін `@capacitor/assets`).

### Підпис релізу для Google Play
```bash
keytool -genkey -v -keystore release.jks -keyalg RSA -keysize 2048 \
        -validity 10000 -alias dragonblast
# вкажіть keystore у android/app/build.gradle (signingConfigs) та зберіть bundleRelease
```
Завантажте отриманий `.aab` у Google Play Console.

## 📺 Підключення реальної реклами (AdMob)

Демо-реклама вже працює. Для реальної:

```bash
npm install @capacitor-community/admob
```
Реалізуйте глобальний міст `window.AdMobBridge.showRewarded(onReward)` —
`ui.js` автоматично використає його замість симуляції (див. `UI.watchAd`).

## 🎮 Як грати

1. Свайпніть кристал у бік сусіднього, щоб поміняти місцями та зібрати 3+ в ряд.
2. Збіги дають очки й **енергію** — стежте за смужками драконів унизу.
3. Коли смужка заповнена — дракон спрацьовує автоматично.
4. Виконайте ціль рівня за відведену кількість ходів, щоб отримати до ★★★.
5. На **острові** витрачайте енергію, щоб вивести нових драконів, та прокачуйте їх.
